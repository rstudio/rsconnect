# rsconnect: Deploy Docs, Apps, and APIs to 'Posit Connect', 'shinyapps.io', and 'RPubs'

Programmatic deployment interface for 'RPubs', 'shinyapps.io', and
'Posit Connect'. Supported content types include R Markdown documents,
Shiny applications, Plumber APIs, plots, and static web content.
